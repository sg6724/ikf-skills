# Instagram Hygiene Checklist

## Profile Essentials

### Basic Setup
- [ ] Username is memorable and searchable
- [ ] Name field (not username) includes brand + keywords
- [ ] Account type is set to "Business" or "Creator"
- [ ] Category is selected
- [ ] Business email and phone added
- [ ] Link to Facebook Page (if applicable)

### Visual Identity
- [ ] **Profile Picture**
  - 320 x 320 pixels minimum (displays at 110 x 110)
  - Recognizable logo or brand mark
  - Clear at small sizes
  - Consistent with other platforms

### Bio (150 characters max)
- [ ] **First 125 characters** are impactful (truncated in some views)
- [ ] Clearly explains who you are/what you do
- [ ] Includes brand personality or unique angle
- [ ] Uses emojis strategically (if on-brand)
- [ ] Contains relevant keywords/hashtags

### Link in Bio
- [ ] Primary website link added
- [ ] Consider link aggregator (Linktree, etc.) for multiple links
- [ ] Link is trackable (UTM parameters) if measuring traffic
- [ ] Link is mobile-optimized

### Highlights
- [ ] Story Highlights added (3-5 key topics)
- [ ] Highlight covers are branded/consistent
- [ ] Cover images match brand aesthetic
- [ ] Titles are clear and concise
- [ ] Content in highlights is up-to-date

### Contact Options
- [ ] Email button enabled
- [ ] Phone button enabled (if applicable)
- [ ] Directions enabled (if physical location)
- [ ] Action buttons utilized ("Shop", "Book", etc.)

### Feed Aesthetics
- [ ] Grid has visual consistency (color palette, style)
- [ ] First 6-9 posts represent brand well
- [ ] Profile preview looks cohesive
- [ ] Posts are high-quality images/videos

## Content Red Flags

- Bio is vague or generic ("We are a company...")
- No link in bio or broken link
- Empty Highlights section
- Low-quality profile picture (blurry, pixelated)
- Inconsistent visual style in grid
- Grammar/spelling errors in bio
- Too many emojis or hashtags in bio
- Contact buttons not enabled
