# Facebook Hygiene Checklist

## Page Essentials

### Basic Information
- [ ] Page name is accurate and searchable
- [ ] Category is set correctly (1 primary, up to 2 additional)
- [ ] Username/vanity URL is claimed (@companyname)
- [ ] Short description (155 characters) is compelling
- [ ] Website URL is added

### Visual Identity
- [ ] **Profile Picture**
  - 180 x 180 pixels minimum (displays at 170 x 170)
  - Clear logo or brand mark
  - Works at thumbnail size
  - Square format

- [ ] **Cover Photo**
  - Dimensions: 820 x 312 pixels (desktop), 640 x 360 (mobile)
  - Brand-aligned, high quality
  - Clear message or visual identity
  - Not text-heavy (avoid violating 20% text rule)

### About Section
- [ ] **Short Description** (155 chars)
  - Compelling hook
  - Clear value proposition
  - Searchable keywords

- [ ] **Long Description** (full "About")
  - Company story/mission
  - What you do and who you serve
  - Unique selling points
  - 2-4 paragraphs
  - Ends with call-to-action

### Contact & Location
- [ ] Phone number
- [ ] Email address
- [ ] Website URL
- [ ] Physical address (if applicable)
- [ ] Hours of operation (if applicable)
- [ ] WhatsApp/Messenger contact enabled

### Additional Setup
- [ ] Call-to-action button configured (e.g., "Learn More", "Shop Now", "Contact Us")
- [ ] Page tabs organized (prioritize important sections)
- [ ] Privacy settings reviewed
- [ ] Roles and permissions set correctly

## Content Red Flags

- Generic "Welcome to our page" copy
- Incomplete contact information
- Low-quality or stretched cover images
- Missing call-to-action button
- Inconsistent branding across platforms
- Outdated pinned post or announcement
