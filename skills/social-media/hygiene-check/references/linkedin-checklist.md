# LinkedIn Hygiene Checklist

## Profile Essentials

### Company Page Basics
- [ ] Company name is accurate and searchable
- [ ] Industry is set correctly
- [ ] Company size is specified
- [ ] Headquarters location is filled
- [ ] Website URL is added

### Visual Identity
- [ ] **Logo/Profile Picture**
  - High resolution (300x300px minimum)
  - Clear, professional, brand-aligned
  - Square format, works at small sizes
  
- [ ] **Cover Image**
  - Dimensions: 1584 x 396 pixels
  - Clear value proposition or brand essence
  - High quality, not pixelated
  - Updated seasonally or for campaigns

### Tagline/Headline
- [ ] Concise (120 characters max)
- [ ] Communicates core value proposition
- [ ] Uses keywords for discoverability
- [ ] Professional and compelling

### About Section
- [ ] **First 156 characters** are compelling (visible without "see more")
- [ ] Clearly explains what the company does
- [ ] Highlights unique value proposition
- [ ] Includes relevant keywords for SEO
- [ ] 2-3 paragraphs maximum
- [ ] Call-to-action at the end (website link, contact info)
- [ ] Professional tone matching brand voice

### Contact Information
- [ ] Phone number
- [ ] Email address
- [ ] Website link
- [ ] Company address

### Additional Features
- [ ] Specialties listed (up to 20)
- [ ] Featured media/content showcased
- [ ] Custom button enabled (e.g., "Visit website", "Contact us")
- [ ] Life/Culture tab populated (if applicable)

## Content Red Flags

- Outdated information (founded year, employee count)
- Generic/"lorem ipsum" placeholder text
- Broken links or outdated URLs
- Inconsistent branding with other platforms
- Missing call-to-action
